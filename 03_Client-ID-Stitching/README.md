# Сквозная идентификация пользователя

### Решаемая проблема

Стандартные формы платформы не передают Google Client ID и Yandex Metrika User ID. Действия пользователя не связать с лидом в CRM.

### Решение

Извлечение, форматирование и добавление _ga и _ym_uid из cookies в скрытые поля формы при отправке.

### Результат

Менеджер может отследить действия пользователя.

### Инструменты

* JavaScript 
* Creatium API
