# Интеграция с платежной системой AnyPay

### Решаемая проблема

Клиенту нужен прием платежей через AnyPay.

### Решение

Разработка клиентской интеграции с AnyPay.
* Извлечение необходимых данных для двух сценариев: покупка с карточки товара и оплата через форму в корзине.
* Генерация контрольной суммы с помощью функций, предоставленных AnyPay.
* Перенаправление пользователя на сформированный URL.

### Инструменты

* JavaScript 
* Creatium API
