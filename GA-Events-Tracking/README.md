# GA Events Tracking: автоматизация аналитического учета

### Решаемая проблема

Для сбора данных об эффективности маркетинга требовалось ручное добавление кода отслеживания событий на каждый интерактивный элемент (формы, кнопки, ссылки). Это было долго, не масштабировалось и постоянно ломалось при изменениях дизайна и обновлениях платформы.

### Решение

Разработано решение, которое автоматизирует привязку целей Google Analytics к элементам DOM.
* Централизация: ключевые цели объявляются в одном месте (массив gaGoals).
* Динамическая привязка: скрипт ищет в DOM элементы с указанными CSS-классами (на основе goal.action) и привязывает к ним обработчики событий.
* Универсальность: скрипт определяет, привязана ли цель к событию отправки формы или к простому клику, обеспечивая корректное отслеживание обоих типов конверсий.

### Результат

Учет GA-событий становится независимым от ручного вмешательства, а аналитика — надежной.

### Подход

Скрипт использует декларативный подход: пользователь просто объявляет цель и добавляет соответствующий CSS-класс в конструкторе, а скрипт берет на себя всю логику отслеживания. API платформы гарантирует, что цели будут привязаны даже к элементам, загружающимся динамически.

### Технологии

* JavaScript
* Creatium API
* Google Analytics (GTAG)

### Концепции

* Отслеживание событий (Event Tracking)
* Автоматизация аналитики
