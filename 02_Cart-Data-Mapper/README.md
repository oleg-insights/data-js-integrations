# Сбор и подготовка информации о товаре

### Решаемая проблема

Стандартные формы платформы не передают в корзину выбранные клиентом опции товара, менеджеры вынуждены уточнять эту информацию у клиентов.

### Решение

Скрипт собирает данные из полей формы и объединяет их в одну стилизованную (разделители, переносы, выделение) строку, после чего передает в корзину.

### Результат

Продавец получает полный набор данных о заказе, а покупатель - красивое отображение опций в корзине.

### Инструменты
* JavaScript
* Creatium API
