# Автоматическое отслеживание событий (Google Tag)

### Решаемая проблема

Функционал отслеживания событий ломался при некоторых изменениях в DOM, пользователям приходилось редактировать код вручную.

### Решение

Автоматическая привязка целей GA к элементам DOM.
* Цели объявляются в одном месте (массив gaGoals).
* Скрипт ищет элементы с указанными CSS-классами (на основе goal.action) и привязывает к ним обработчики.
* Скрипт определяет, привязана ли цель к событию отправки формы или к простому клику.

### Результат

Учет событий не зависит от ручного вмешательства.

### Инструменты

* JavaScript
* Creatium API
* GTAG
