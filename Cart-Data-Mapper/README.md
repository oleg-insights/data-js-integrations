# Cart Data Mapper: сбор и подготовка информации о товаре

### Решаемая проблема

Недостаток данных в заказах (стандартные формы платформы не передают в корзину выбранные клиентом опции товара). Это вынуждало менеджеров вручную уточнять детали, замедляя работу.

### Решение

Скрипт выполняет Data Mapping в браузере. Он собирает разрозненные данные из полей формы и объединяет их в одну, стандартизированную и стилизованную (разделители, переносы, выделение) строку, после чего передает данные в корзину.

### Результат

Продавец получает полный набор данных о заказе мгновенно. Вторичная польза - удобное отображение опций в корзине для покупателя.

### Подход

Скрипт использует API платформы для устойчивости к изменениям в разметке (DOM) и соответствия стандартам компании.

### Технологии
* JavaScript
* Creatium API

### Концепции
* Data mapping
* Трансформация данных
* Агрегация данных
